# 🚀 Financely - Gestão Financeira Pessoal

O **Financely** é um ecossistema completo para controle financeiro, permitindo que usuários gerenciem transações, categorias e metas de economia em uma interface moderna e intuitiva.

## 🛠️ Stack Tecnológica

- **Frontend:** Next.js 14, Tailwind CSS, Lucide React.
- **Backend:** Node.js, Express, JWT (Autenticação), Bcrypt.
- **Banco de Dados:** PostgreSQL (Neon Tech).
- **Deploy:** Vercel (Client) e Render (API).

## ✨ Funcionalidades Implementadas

- **Autenticação:** Sistema de login e registro seguro com hash de senha.
- **Transações:** Fluxo completo de Entradas e Saídas com balanço automático.
- **Categorias:** Organização por tipos de gastos (Padrão e Personalizadas).
- **Sistema de Metas:** - Criação de objetivos financeiros com prazos.
  - Barra de progresso dinâmica baseada no valor poupado.
  - Função de aporte (Poupar) e exclusão de metas.
- **Relatórios:** Exportação de dados para PDF.
